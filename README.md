# # The Last of Us Website

Projeto desenvolvido como atividade prática do curso de Sistemas de Informação.

O objetivo deste projeto é criar um website informativo inspirado no primeiro jogo da franquia The Last of Us, utilizando HTML, CSS e JavaScript. O site será estruturado para apresentar conteúdos relacionados à história, personagens, curiosidades e ambientação do jogo, permitindo que pessoas que ainda não conhecem o universo de The Last of Us possam entender melhor sua proposta.

---

## 📌 Sobre o Projeto

Este website será desenvolvido com foco em praticar conceitos fundamentais de desenvolvimento web.

A proposta é criar um website informativo inspirado no universo de The Last of Us, contendo páginas explicativas, conteúdo sobre o jogo e uma tela de cadastro de usuários.

---

## ⚙️ Funcionalidades Planejadas

As funcionalidades previstas incluem:

* Página inicial temática
* Tela de cadastro de usuários
* Navegação entre páginas
* Páginas informativas sobre o jogo
* Interface inspirada no universo pós-apocalíptico

---

## 🛠️ Como Executar o Projeto

### Clonar o Repositório

acesse o terminal de seu computador, caso seja linux apenas faça o que está abaixo:
git clone https://github.com/seu-usuario/nome-do-repositorio.git.
dentro de um diretório de sua prefência (Geralmente em Documentos).
Caso esteja utilizando windowns será necessário a instalação do git bash, após a instalação, apenas faça as instruções dadas acima.

### Abrir o Projeto

Após clonar, basta abrir o arquivo `index.html` em um navegador de sua preferência.
